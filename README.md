# acceptTI 科研人格测试

基于 SBTI 改编的科研版人格测试，专为学术从业者设计。

**在线访问**: https://accept-ti.pages.dev/

## 什么是 acceptTI

acceptTI (Academic Character Research Type Indicator) 是一个娱乐性质的科研人格测试：

- **31道题目** 覆盖科研自我认知、情绪调节、价值取向、执行风格、人际关系五大维度
- **27种人格类型** 包括25种标准人格、1种兜底人格和1种隐藏人格
- **15维度评分** 详细解读你的学术性格

## 人格类型一览

### 学术地位类
- **PHD** - 学术难民：签证要过期了，论文还没accept
- **POST** - 博后流浪汉：三年又三年，下一个站在哪
- **TENURE** - 终身咸鱼：已拿铁饭碗，但是碗是漏的
- **AP** - 助理教授：非升即走，每天都在走钢丝
- **MASTER** - 硕士混子：能毕业就行，别的无所谓
- **UG** - 本科科研炮灰：进实验室的时候雄心壮志...
- **GHOST** - 实验室幽灵：每天都在，但没人知道我在干嘛

### 科研行为类
- **CTRL** - 控制狂魔：你的实验记录本，我要逐字检查
- **PAPER** - 论文永动机：reject是常态，submit才是生活
- **DATA** - 数据奴隶：跑了一周，发现原始数据错了
- **GRANT** - 本子乞丐：每年这个时候都在求国家给钱
- **IDEA** - 点子批发商：idea很多，能做的一个没有
- **CTRLV** - 学术裁缝：这缝缝，那补补，又一篇
- **READ** - 文献收藏家：Zotero里3000篇，看过的30篇
- **CODE** - 调参炼丹师：这个learning rate不行，再试试
- **PPT** - 汇报表演艺术家：图要好看，故事要圆，数据不重要

### 社交关系类
- **BOSS** - 老板心腹：老板说的都对，老板永远正确
- **SOLO** - 学术独狼：合作？不存在的，都是拖累
- **NET** - 社交蝴蝶：coffee break比我实验还重要
- **REVIEW** - 匿名恶毒侠：reject with prejudice，心情好点了
- **COAUTH** - 挂名狂魔：通讯作者？给我加上，我认识主编

### 心理状态类
- **DEAD** - 学术尸体：心如死灰，但还在投简历
- **HOPE** - 盲目乐观者：这篇肯定中，上次也是这么说的
- **FOMO** - 同辈焦虑症患者：他发Nature了，我还在改introduction
- **BURN** - 燃尽综合征：曾经热爱，现在只想睡觉

### 特殊人格
- **HHHH** - 科研傻乐者：啥都不懂，但每天挺开心的（兜底人格）
- **OVERNIGHT** - 通宵战神：日出而作，日出而息（隐藏人格）

## 技术实现

- 纯原生 HTML/CSS/JavaScript，单文件架构
- 15维度曼哈顿距离匹配算法
- 响应式设计，支持移动端
